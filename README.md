# `yarn-mode`

This package contains a major mode for `yarn.lock` files.

## Installation

No MELPA just yet. Copy `yarn-mode.el` to your load path and then:

``` emacs-lisp
(require 'yarn-mode)
```
Yes. It's that simple.
